# 快速开始指南

## 第一次使用

### 1. 运行程序

**从源码运行:**

```bash
python main.py
```

程序窗口将会打开。

### 2. 选择测试照片目录

在"目标目录"区域：

1. 点击"浏览..."按钮
2. 选择 `test_photos` 目录
3. 确保"包括子目录"已勾选

### 3. 尝试不同的修改模式

#### 模式1: 当前时间

1. 选择"修改为当前时间"
2. 点击"开始修改"
3. 确认操作
4. 查看日志输出

#### 模式2: 指定时间

1. 选择"修改为指定时间"
2. 使用日期时间选择器设置：
   - 年: 2026
   - 月: 03
   - 日: 04
   - 时间: 15:30:00
3. 点击"开始修改"
4. 确认操作

#### 模式3: 相对时间调整

1. 选择"相对时间调整"
2. 设置偏移量，例如：
   - +3 天
   - +2 小时
   - +30 分钟
3. 点击"开始修改"
4. 确认操作

### 4. 查看结果

- **日志窗口**: 实时显示处理进度
  - 绿色文字: 成功
  - 红色文字: 失败
  - 橙色文字: 警告

- **完成统计**: 处理完成后会弹出摘要对话框

### 5. 验证修改

#### 方法1: 文件资源管理器

1. 打开 test_photos 目录
2. 右键点击照片 → 属性 → 详细信息
3. 查看"拍摄日期"是否已修改

#### 方法2: 使用ExifTool（如果已安装）

```bash
exiftool test_photos/1.jpg
```

**方法3: 再次运行程序**
程序日志会显示当前文件时间

## 常见使用场景

### 场景1: 批量修正相机时间错误

假设相机时间快了1天：

1. 选择照片目录
2. 选择"相对时间调整"
3. 设置: -1 天, 0 小时, 0 分钟
4. 开始修改

### 场景2: 统一活动照片时间

将所有照片设置为活动开始时间（2026-03-04 10:00:00）：

1. 选择照片目录
2. 选择"修改为指定时间"
3. 设置日期时间为: 2026年3月4日 10:00:00
4. 开始修改

### 场景3: 更新扫描文档时间

将扫描的照片设置为当前时间：

1. 选择照片目录
2. 选择"修改为当前时间"
3. 开始修改

## 日志功能

### 导出日志

1. 处理完成后，点击"导出日志"
2. 选择保存位置
3. 日志会保存为文本文件

### 清空日志

点击"清空日志"按钮清空日志窗口

## 注意事项

⚠️ **重要提醒:**

- test_photos中的文件是用于测试的
- 正式使用前，请先备份重要照片
- 修改操作不可撤销

## 打包为可执行文件

如果要创建独立的exe文件：

```bash
# 确保已安装所有依赖
pip install -r requirements.txt

# 使用PyInstaller打包
python -m PyInstaller PhotoTimeTool.spec

# 可执行文件位于 dist/PhotoTimeTool.exe
```

详细打包说明请参考 [BUILD.md](BUILD.md)
